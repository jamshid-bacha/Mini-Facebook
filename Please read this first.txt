
Project Flow

1) The user will register him/herself
2) during registration if any field is empty JavaScript code will alert that please input 
something in that specific field and JS code will focus on that field.
3) If the password is less than 8 characters, the JS code will alert
4) If the password does not match the JS code will alert
5) If personID is already taken then the message will display
6) If the email is already taken then the message will display 
7) If everything is fine then the message will display that now you can log in and register successfully
8) If the email and password do not match during login message will appear that the email or password 
is incorrect
9) If user login then the image will display options (send message, inbox, outbox, and logout)
